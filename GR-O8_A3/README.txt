To run the program,
1. We have to change into the directory cotaining the files.
2. First we have to run the command : python3 work_server.py
3. The previous command initiate 3 work server
4. Then we have to run the command : python3 server.py ,which will initial the server.
5. Lastly we have the client initiating command : python3 client.py
6. The clients will send number in range 50 and the server will send the numbers to work_server to calculate the square value of the numbers.
7. The work_server will send the calculated values to the server and the server will forward the values to the clients.
