First, the server should be initiated. It can be done by executing the following steps.
1. Change into the directory containing the server.py file. 
2. Run this command: python server.py 

Then you can connect the client by executing the following steps.
1. Change into the directory containing the client.py. 
2. Run this command: python client.py

Nota Bene: 
1. You need to change the port if it is not killed properly. 
2. You have to set the port and host addresses properly.
